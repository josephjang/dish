
dish - ssh/rsh command utility for the distributed environment
==============================================================

*dish* is a utility to execute commands in remote hosts.

Synopsis
--------

```
dish -c COMMAND HOST_SPEC
```

Example
--------

```
$ dish -c uptime foo1,foo2~foo5
```
